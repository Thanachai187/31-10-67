Array เก็บข้อมูลประเภทเดียวกัน
 แบบที่จัดเรียงต่อเนื่องในหน่วยความจำ
 มีขนาดคงที่ 
 เเบบระบุ index

Linked List 
 แต่ละโหนดจะมีค่าข้อมูลและตัวชี้ไปยังโหนดถัดไป
 ใช้สำหรับการแทรกและลบข้อมูลบ่อย ๆ 
 เนื่องจากมีการจัดการที่ยืดหยุ่นกว่าอาเรย์
 Doubly Linked List
 โหนดแต่ละโหนดมีตัวชี้ไปยังโหนดถัดไป
 และโหนดก่อนหน้า

Stack LIFO ล่าสุดที่ถูกเพิ่มจะถูกนำออกก่อน


Queue FIFO ถูกเพิ่มเข้ามาก่อนจะถูกนำออกก่อน

Tree 
 โครงสร้างข้อมูลที่มีโหนดหลัก (root) และโหนดย่อย (child) 
 ซึ่งจัดเรียงในรูปแบบที่มีความสัมพันธ์กัน
 และการแสดงข้อมูลที่มีความสัมพันธ์
 Binary Tree
  ต้นไม้ที่มีโหนดสูงสุด 2 โหนดในแต่ละระดับ
 Binary Search Tree (BST)
  ซ้ายมีค่าต่ำกว่าโหนดหลัก
  และโหนดขวามีค่าสูงกว่าโหนดหลัก

Graph ประกอบด้วยจุด (vertex) 
และเส้นเชื่อม (edge) ที่เชื่อมโยงระหว่างจุด
